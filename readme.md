# Basic C Compiler
BCC is a basic C compiler.

## Supported functions
* **Athm**: Addition
* **Athm**: Soustraction
* **Athm**: Multiplication
* **Athm**: Division

* **Athm**: Plus égal
* **Athm**: Moins égal

* **Fctn**: Print
