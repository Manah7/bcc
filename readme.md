# Basic C Compiler
BCC is a basic C compiler.

## Examples
```
$ make
$ ./bcc < example/4example.c 2>/dev/null
main:
1	AFC 4 2 
2	COP 0 4 
3	AFC 8 2 
4	EQU 12 0 8
5	JMF 12 7 0
6	PRI 0 
7	AFC 20 9 
8	COP 16 20
```

## Supported functions
* **Athm**: Addition
* **Athm**: Soustraction
* **Athm**: Multiplication
* **Athm**: Division

* **Athm**: Plus égal
* **Athm**: Moins égal

* **Fctn**: Print
* **Loop**: If
